# SpongeBobRoleSelect

移动端网页项目骨架，技术栈为 `React + Vite + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion`。

## 当前状态

- 已创建项目目录与基础文件。
- 已配置 Tailwind Vite 插件。
- 已配置 `@/*` 路径别名。
- 已加入 `components.json`、`Button` 和 `cn`，可继续用 shadcn/ui CLI 扩展组件。
- 首页目前是移动端空白壳子，方便后续直接接内容。

## 启动前需要安装

当前目录还没有安装依赖。联网后在项目目录执行：

```bash
pnpm install
```

## 本地开发

```bash
pnpm dev
```

## 说明

- 官方 Motion for React 现在通过 `motion` 包安装，并从 `motion/react` 导入。
- 当前首页文件在 [src/App.tsx](D:\dev\code\SpongeBobRoleSelect\src\App.tsx)。
